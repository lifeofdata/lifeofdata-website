== Description ==

This plugin has support for (a single) widget per post/page and unlimited number of Audiboo embeds using the shortcode.

Widget usage:

* Add URL to Audioboo widget panel in your blog post.

Shortcode usage:

* [audioboo url="http://audioboo.fm/boos/306812-radio-2"]
